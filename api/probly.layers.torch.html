reset_parameters()[source]¶

Reset the parameters of the Bayesian conv2d layer.

Setting a=sqrt(5) in kaiming_uniform is the same as initializing with :rtype: None

uniform(-1/sqrt(k), 1/sqrt(k)), where k = weight.size(1) * prod(*kernel_size) For more details see: https://github.com/pytorch/pytorch/issues/15314#issuecomment-477448573

Return type:

None

reset_parameters()[source]¶

Reset the parameters of the NormalInverseGamma layer.

Setting a=sqrt(5) in kaiming_uniform is the same as initializing with uniform(-1/sqrt(in_features), 1/sqrt(in_features)). For details, see https://github.com/pytorch/pytorch/issues/57109.
